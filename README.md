# Pancrepal 小胰宝 - 病情主动管理助手

## 项目简介
小胰宝 (Pancrepal) 是一款专为肿瘤病友及其家属设计的非营利性科普与病情主动管理平台。通过结构化的管理路线图、游戏化的知识学习、互助社区以及基于 Gemini AI 的智能助手，赋能患者，消除信息差。

## 核心功能
- **智能管理路线图**：结构化展示临床治疗、营养、心理等关键节点，提供一页纸式的管理路径。
- **知识大闯关**：寓教于乐，通过答题赢取 AI 互动配额，激励主动学习。
- **AI 科普助手**：基于 Gemini-3-Flash 驱动，支持文本即时咨询。
- **实时语音助理**：基于 Gemini Live API 的真人感通话体验，支持多音色切换与实时打断。
- **互助社区**：病友间真实的康复经验分享，传递温暖与希望。
- **适老关怀模式**：全局字体放大，UI 组件圆润化，简化交互逻辑，服务高龄患者。

## 🎨 视觉与体验特色
- **大圆形 AI 交互**：采用醒目的圆形品牌 Logo 入口，模拟自然的人机对话体验。
- **呼吸灯动效**：实时通话状态下具备柔和的多层光环动效，提供直观的状态反馈。
- **极致简洁**：针对移动端优化，核心功能触手可及。

## 🚀 部署注意事项

如果您计划将本项目部署到静态托管平台，请务必在 `index.html` 中正确引用入口脚本：

```html

  <div id="root"></div>

```

## 技术栈
- **Frontend**: React 19, Tailwind CSS, Lucide React
- **AI**: Google Gemini API (@google/genai)
- **Engine**: Gemini Live API (Native Audio)
